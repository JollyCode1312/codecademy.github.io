CSS properties

Property Description
@media query A media query in CSS is used to apply different styles to a web page based on certain conditions, such as the device's screen size, orientation, resolution, or other media features. Media queries allow you to create responsive designs that adapt to different devices and screen sizes, providing a better user experience across various devices like desktops, tablets, and mobile phones.
